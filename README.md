# Abstarct
Road safety enforcement calls for effective monitoring systems that can spot traffic rule infractions, including riding without a helmet. Using deep learning methods, this study proposes an image-based automatic helmet and licence plate detection system. The suggested method uses the YOLO (You Only Look Once) object detection method to detect number plates and helmets from incoming photos. One YOLO model is trained for helmet detection, the other for licence plate identification; two independently trained YOLO models are utilised. Once a license plate is found, the system extracts the region of interest and uses Optical Character Recognition (OCR) with EasyOCR to get the vehicle registration number. The output image is annotated with extracted plate text, bounding boxes, identification labels, and helmet condition (Helmet Present / No Helmet). Python, OpenCV, and Ultralytics YOLO framework are used to implement the system. Experimental data show that the suggested technique is appropriate for smart surveillance applications and traffic monitoring since it offers exact detection and vivid violation depiction.
# Introduction
Still, a leading source of injuries and deaths worldwide is road traffic accidents. Not following safety laws, particularly not wearing helmets when riding motorbikes, greatly aggravates these events . Manual monitoring of traffic offences consumes time, is labour-intensive, and is prone to human mistake. For traffic control and enforcement, therefore, intelligent automated systems based on deeplearning and computer vision are progressively employed. Particularly, convolutional neural networks (CNNs) and more recent breakthroughs in deep learning have dramatically raised object detection system accuracy. Among these, YOLO (You Only Look Once) is well-known for its high accuracy and real-time detection abilities one-pass detection helps YOLO fit Real-time users, including traffic management.
Using two independently trained YOLO models, this initiative creates an image-based helmet and license plate recognition system. The first model identifies helmets and determines if a rider is wearing one. The second model finds the vehicle's license plate. EasyOCR extracts the vehicle registration number via Optical Character Recognition (OCR) after finding the license plate.
The system notes the results with bounding boxes, detection labels, helmet status, and license plate number after scanning an input image for pertinent objects. Python, OpenCV, the Ultralytics YOLO framework, and Easyocr make up the implementation.
The suggested system shows how artificial intelligence can automatically detect traffic infractions, lower human involvement, and raise law enforcement effectiveness. This strategy can be seen as a first stride toward intelligent traffic management and smart surveillance.
